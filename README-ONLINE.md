# تشغيل برنامج مصاريف العائلة على الإنترنت

## الملفات المطلوبة

ارفع هذه الملفات إلى GitHub أو إلى الاستضافة:

- `family-expenses.html`
- `family-expenses-server.mjs`
- `package.json`
- `family-expenses-data.json` إذا كان موجودًا وتريد نقل البيانات الحالية

## الطريقة المقترحة: Render

1. افتح حساب في Render.
2. أنشئ مستودع GitHub جديد وارفع ملفات البرنامج إليه.
3. في Render اختر `New Web Service`.
4. اربط مستودع GitHub.
5. الإعدادات:
   - Runtime: Node
   - Build Command: اتركه فارغًا أو اكتب `npm install`
   - Start Command: `npm start`
6. بعد النشر سيعطيك Render رابطًا مثل:
   `https://family-expenses.onrender.com`

## مهم جدًا لحفظ البيانات

البرنامج يحفظ البيانات في ملف:

`family-expenses-data.json`

على بعض الاستضافات قد تضيع الملفات عند إعادة التشغيل أو إعادة النشر إذا لم تضف تخزينًا دائمًا.

في Render أضف Persistent Disk أو استخدم قاعدة بيانات لاحقًا إذا أردت نظامًا أقوى.

## بعد النشر

- الأب يدخل من رابط الإنترنت الجديد.
- الأبناء يدخلون من نفس الرابط.
- لا يحتاجون أن يكونوا على نفس الواي فاي.
- الكمبيوتر في البيت لا يحتاج أن يبقى شغالًا إذا كان البرنامج مرفوعًا على الاستضافة.

## تنبيه أمان

قبل مشاركة الرابط خارج العائلة:

- غيّر كلمة مرور الأب.
- ضع كلمة مختلفة لكل شخص.
- لا تشارك الرابط مع أي شخص خارج العائلة.
